# song-pronunciation-kor

## 프로젝트 개요

영어 노래 가사를 한글 발음 기호로 변환하여 한국인이 쉽게 따라 부를 수 있도록 돕는 웹 애플리케이션입니다.

### 핵심 아이디어

- 영어는 글로벌 언어이지만 한글과 구조가 달라 발음하기 어려운 부분이 있음
- 단순히 단어나 문장을 한글 발음 기호로 표기하면 외우는 데 시간이 걸림
- 노래를 통해 자연스럽게 영어 발음을 익힐 수 있도록 영어 노래 가사를 한글 발음 기호로 변환

### 주요 기능

- 영어 노래 가사를 한글 발음 기호로 변환 (텍스트 기반)
- 노래 재생 기능은 불포함 (저작권 이슈 고려)
- 노래 검색 기능
  - 토글 버튼으로 제목 검색 / 아티스트 검색 선택
  - 검색 결과 리스트: 노래 제목, 아티스트명, 앨범명, 작곡가명, 작사가명 표시
  - 검색 결과 클릭 시 노래 가사 페이지로 이동
- 가사 데이터 소스: 외부 API 또는 웹 크롤링 활용 (로컬 DB 미사용)

## 개발 가이드라인

### 시작하기

1. 프로젝트 디렉토리로 이동
2. 로컬 서버 실행 (예: `python3 -m http.server 8000`)
3. 브라우저에서 `http://localhost:8000` 접속

### 핵심 구현 사항

1. **노래 검색 기능**
   - 토글 버튼: 제목 검색 / 아티스트 검색 전환
   - 외부 API 또는 웹 크롤링으로 가사 데이터 가져오기
   - 검색 결과 리스트 표시
     - 노래 제목
     - 아티스트명
     - 앨범명
     - 작곡가명
     - 작사가명

2. **영어 → 한글 발음 변환 알고리즘**
   - 영어 발음 규칙을 기반으로 한글 음소로 매핑
   - 예외 처리 (불규칙 발음, 연음 등)

3. **페이지 라우팅**
   - 검색 페이지 (index.html)
   - 가사 표시 페이지 (lyrics.html)
   - URL 파라미터로 선택한 노래 정보 전달

4. **가사 표시**
   - 원문(영어)과 발음 기호(한글)를 나란히 표시
   - 가독성 좋은 레이아웃
